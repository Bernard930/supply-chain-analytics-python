# 📊 Ghana Trade Flow Insights (2022)

This project provides a data-driven exploration of Ghana's import and export trends for the year 2022. Using Python and Seaborn, the notebook visualizes key trade flows to offer insights into the country’s economic activity and major trading partners.

## 🔍 Objective

- Analyze Ghana's trade data to uncover patterns in imports and exports.
- Visualize top trading countries and product categories.
- Provide actionable insights through clean, engaging charts.

## 🛠️ Tools & Libraries Used

- **Python**
- **Google Colab**
- **Pandas** – data cleaning & transformation
- **Seaborn** – statistical data visualization
- **Matplotlib** – custom plot formatting
